This is a demo flask application for python flask tutorial. To run this application, follow the steps below:

1. Create a virtual environment:
   python3 -m venv .venv
2. Activate the virtual environment:
   source .venv/bin/activate
3. Install the required packages:
   pip install -r requirements.txt
4. Run the application:
   python app.py
5. Open your browser and go to http://localhost:5500 to see the application running.
6. To deactivate the virtual environment, simply run:
   deactivate

   Cheers~~!
